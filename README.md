# Hack n' Slash

Pagina estatica sobre protagonistas de videojuegos de accion.

## Despliegue automatico en GitHub Pages

1. Crea un repositorio en GitHub y sube estos archivos.
2. En GitHub, ve a `Settings > Pages`.
3. En `Build and deployment`, selecciona `GitHub Actions`.
4. Haz push a `main` o `master`.
5. El workflow `Deploy static site to GitHub Pages` publicara la pagina automaticamente.

## Editar imagenes

La imagen principal esta en `assets/hero-protagonistas.png`.

Las imagenes de las tarjetas estan en `assets/characters/`. Puedes reemplazarlas por imagenes propias o URLs externas cuando quieras.
